# SUPERSTORE-SALES-DASHBOARD

🛒 Superstore Sales Dashboard (Power BI)

📌 Project Overview

This project presents an interactive **Sales Dashboard** built using Power BI to analyze retail performance and forecast future sales trends.
The goal is to derive actionable insights and support data-driven decision-making.


🎯 Business Objective

* Track overall sales and profit performance
* Identify high and low-performing categories
* Analyze regional trends
* Understand discount impact on profit
* Forecast future sales trends


📊 Key Metrics

* 📈 Total Sales: **1.57M**
* 💰 Total Profit: **175K**
* 📊 Profit Margin: ~11%



📊 Key Insights

* 🏆 **Technology category** is the most profitable
* 📉 High discounts are reducing overall profit
* 🌍 **West region** generates the highest sales
* ⚠️ Some sub-categories show **high sales but low/negative profit**
* 📅 Sales trends show growth but profit is inconsistent


🔮 Forecast Analysis

* Implemented **time-series forecasting** using Power BI
* Predicted future sales based on historical trends
* Identified expected growth patterns and seasonal variations
* Helps business in planning inventory and sales strategy



📌 Dashboard Features

* KPI Cards (Sales, Profit, Profit Margin)
* Region-wise Analysis
* Category & Sub-category Breakdown
* Monthly Sales Trends
* Discount vs Profit Analysis
* 🔮 Sales Forecasting Page
* Interactive Filters (Region, Category, Date)



🛠️ Tools Used

Power BI – Data visualization & forecasting



📂 Dataset

* Superstore Sales Dataset
* Includes Orders, Sales, Profit, Discount, Region, Category



⚙️ Steps Performed

1. Data cleaning using Power Query
2. Data modeling
3. Created DAX measures (Sales, Profit, Margin)
4. Built dashboard visuals
5. Applied forecasting technique
6. Derived insights & recommendations

* Reduce high discounting in low-profit areas
* Focus on high-profit categories like Technology
* Improve performance of loss-making sub-categories
* Use forecast insights for better planning and decision-making
